# API Películas y Series

Una App web para explorar películas y series usando la API plan gratuito de TMDB.

## Funcionalidades
- Explorar películas y series populares
- Filtrar por género y año
- Paginación
- Diseño responsive con menú hamburguesa

## Tecnologías
- JavaScript (ES Modules)
- SCSS
- Rollup
- TMDB API

## Configuración
1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Copia `.env.example` y renómbralo a `.env`
4. Agrega tu API key de TMDB en `API_KEY`
5. Ejecuta: `npm run build` y `npm run sass`
